# canada-tax-rates

**Canadian freelancer tax rates & thresholds as verified, versioned JSON.**

Published by [freelancertax.ca](https://freelancertax.ca) from the same deterministic,
test-gated engine that powers its calculators. Every file is generated by a build
script that serializes the engine's data exports directly — **never hand-edited** —
and each file carries the date its figures were last verified against CRA /
Revenu Québec published sources. How the engine is verified (hand-computed anchors,
independent cross-checks, randomized sweeps, ~190k+ assertions gating every deploy):
[freelancertax.ca/how-we-verify](https://freelancertax.ca/how-we-verify/).

## Files (also served live at [freelancertax.ca/data/v1/](https://freelancertax.ca/data/v1/index.json))

| File | Contents |
|---|---|
| `data/v1/index.json` | Manifest: version, license, per-file verification dates |
| `data/v1/federal-ontario-2025.json` | 2025 federal + Ontario brackets, BPA + phase-down, surtax, health premium, CPP incl. CPP2, core GST/HST figures |
| `data/v1/federal-ontario-2026.json` | Same structure, 2026 |
| `data/v1/provinces.json` | BC, AB, MB, SK, NS, NB brackets + BPA (2025 + 2026) and Québec (QPP, QPIP, federal abatement) |
| `data/v1/gst-hst.json` | GST/HST registration thresholds, Quick Method rate grids (services + goods) and exclusions, platform rules, export cites |
| `data/v1/instalments.json` | CRA instalment thresholds and due dates |
| `data/v1/deadlines.json` | Canadian tax deadlines (statutory dates; apply the next-business-day rule) with sources |

The live site endpoint is the canonical copy; this repo mirrors it per release.

## Disclaimer

Figures are estimates for planning, not tax or legal advice. Each JSON file embeds
the verbatim disclaimer and its `figures_verified` date. Verify against CRA
published sources before relying on any figure for filing.
